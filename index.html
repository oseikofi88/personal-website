Pet/Hobby Projects

  • Find from my followers who (Twitter Project)
    This project is to find my followers who can help me with a request based on their profile picture, tweets, replies, bio/description and likes
    For example, if I want to buy a car, I can find a follower who is a car dealer or has a car dealership based on the affore mentioned criteria about the follower's information.

    The algorithm runs as follows:
    1. Profile picture: If the profile picture is car or automobile-related, then the follower is a potential car dealer
    2. Tweets: If the follower tweets about cars, then the follower is a potential car dealer
    3. Replies: If the follower replies to tweets about cars, then the follower is a potential car dealer
    4. Bio/Description: If the follower's bio/description contains car dealer, then the follower is a potential car dealer
    5. Likes: If the follower likes tweets about cars, then the follower is a potential car dealer
    The algorithm then ranks the followers based on the number of criteria met and gives the ranking in chronological order

    Tools:
    Integration: Twitter APIs, Google's search APIs
    Backend: Java Springboot
    Framework: Apache Camel
    Hosting: AWS
    AWS Services used: EC2, S3, Lambda, API Gateway, Cloudwatch, SNS, SQS, Route 53, VPC, IAM, Cloudfront, Cloudformation, Rekognition, CodeDeploy, CodePipeline
    CI/CD Tool used: Jenkins
    Enterprise Integration Patterns Used: Splitter, Aggregator, Content-based router, Message filter
    Databases: Redis
    Message broker: RabbitMQ
